# 3-1: Text Manipulation

## Quiz Questions

**How do we easily access the last element in a sequence?**

* The `.last()` method
* The `[-1]` index
* `last(seq)`
* The `[0]` index

**How do we reverse lists in place?**

* The `.reverse()` method
* Slicing by `[::-1]`
* `rev(seq)`
* `.slice(-1)`


**How do we make every letter in a string upper case?**

* `.capitalize()`
* `to_upper()`
* `upper()`
* `uppercase()`
